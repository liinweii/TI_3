# TI_3
## Лабораторная работа 3. Система RSA.
Вариант 1

## О проекте
Программа для шифрования и дешифрования файлов различных расширений и размеров с помощью ключей, сгенерированных на основе системы RSA.

### Ссылка на программу: https://github.com/liinweii/TI_3/blob/main/Project1.exe

# 📘 Инструкция по использованию программы
## Шаг 1. Загрузите файл для шифрования
Выберите, какой файл будет открыт (для шифрования или дешифрования). Нажмите на кнопку "Open" и выберите файл для шифрования.

## Шаг 2. Заполните поля ввода и сгенерируйте оставшиеся знаечния  
Заполните поля ввода p, q и D. Числа должны быть простыми. 
После этого нажмите на кнопку "Count parametrs". Программа сгенерирует остальные значения и выведет их. 

## Шаг 3. Шифрование и дешифрование
Используйте кнопки под полями ввода:

Кнопка "Encrypt":	преобразует исходную последовательность чисел в зашифрованную  
Кнопка "Decrypt":	восстанавливает исходную последовательность чисел из зашифрованной  

## Шаг 4. Сохранение результата
Нажмите "Save": результат сохранится в выбранный вами файл.
